# ProcureFlow Desktop

Single-user desktop application for procurement management. Built with Tauri v2, React, TypeScript, and SQLite.

## Download

Download the latest installer from the [Releases](../../releases) page.

## Features

- Full procurement lifecycle: RFPs, proposals, suppliers, clients, schedules, contracts
- Executive dashboard with charts and KPIs
- Project management with 9 tabs (packages, financials, risks, Q&A, deliveries, etc.)
- Notes system with @mentions and threading
- Excel import/export
- Dark mode
- Built-in update system
- All data stored locally — no cloud dependency

## Technical Details

- **Platform:** Windows 10/11 (x64)
- **Installer size:** ~8 MB
- **Data location:** `%LOCALAPPDATA%\com.procureflow.desktop\`
- **Network:** No internet required (optional update check only)
